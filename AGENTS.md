# AI Agent Rules

이 프로젝트의 AI 작업 규칙은 다음 문서에 정의되어 있다:

- **docs/ai/agent.md** — AI 작업 규칙(필수 규칙)
- **docs/specs/03_functional-spec.md** — 기능 요구사항 명세서

모든 Codex 작업(설계, 구현, 수정, 분석 등)은  
반드시 위 두 문서를 기준으로 판단해야 한다.

프로젝트 내에서 AI를 사용할 때는 다음을 기본 원칙으로 한다:

1. **agent.md**의 규칙을 시스템 프롬프트처럼 항상 유지하고 준수한다.
2. 기능, API, 데이터베이스, 서비스 로직을 설계하거나 수정할 때  
   **03_functional-spec.md** 를 기준으로 판단한다.
3. 새로운 기능 제안이나 코드 변경이 있을 경우  
   두 문서와 충돌하지 않는지 반드시 검증해야 한다.

Codex CLI 또는 Codex 확장을 사용할 때  
추가 지시 없이도 이 문서를 규칙의 근거로 삼는다.
